# Changelog

### Phase 7 — 测试调优、统计系统、文档完善
- 波次平衡：降低敌人类型解锁阈值，更早引入敌人 variety
- 波次缩放：敌人速度每 5 波 +10%，血量每 3 波 +1
- 新增本局统计系统：射击次数、命中次数、命中率、击杀数、金币收集、道具收集、受伤次数、存活时间
- 新增游戏结束详细统计画面（含命中率）
- 新增屏幕震动效果：玩家受击时触发，带强度衰减
- 为 input.js、powerup.js、wave-manager.js、engine.js 补充完整 JSDoc
- 更新 README.md 为最终版本，反映实际实现状态
- 移除未使用的 src/game.js（旧版英雄抓怪物代码）

## [0.6.0] - 2026-05-26

### Phase 6 — 音效系统、存档系统、菜单完善
- 新增 AudioManager：Web Audio API 程序化生成全部音效
  - 射击（噪声衰减）、击中（方波下降）、敌人死亡（噪声）、玩家死亡（锯齿波下降）
  - 道具拾取（琶音 C5-E5-G5）、金币（1200Hz 短音）、购买（双音和弦）、传送（扫频）
  - BGM：低频鼓点循环（每 0.6 秒）
- 新增 localStorage 存档系统：最高分、最高波次、总金币、游戏次数
- 菜单画面显示历史记录（最高分/最高波次/游戏次数）
- 游戏结束画面显示"新最高分"/"新最高波次"标识
- 音频在用户首次交互时初始化，浏览器策略兼容

## [0.5.1] - 2026-05-26

### 紧急修复 — 敌人卡墙 + 重新开始
- 修复敌人卡在凹角（两个相邻障碍）完全无法移动的 bug
  - 添加 stuckCounter：连续 10 帧无法移动时强制重新选择安全目标
  - 进入随机游走模式（uninterested = true）脱离卡死
- 新增 R 键重新开始功能（PLAYING / PAUSED 状态）

## [0.5.0] - 2026-05-25

### Phase 5 — 全部敌人 + HUD + 粒子特效
- 新增 4 种敌人：Ghost（飞行）、Mummy（高血）、Devil（飞行+高速）、Spikey（变身）
- Spikey 特殊机制：移动到随机目标后变身，血量 +5，橙色光环
- Ghost/Devil 飞行物理：惯性移动，无视地形
- 新增每种敌人的专属绘制特征（眼睛、角、斑点、绷带等）
- HUD 完整化：生命值（心形）、命数、金币、得分、敌人数量、波次进度、道具计时器
- 心形图标使用 Canvas bezierCurveTo 绘制

## [0.4.0] - 2026-05-24

### Phase 4 — 商店系统
- 新增 Shop + ShopItem 类
- 每 2 波商人出现，3 选 1 升级
- 商人走入动画（从屏幕上方进入）
- 商品动态生成：基于玩家当前升级状态自动替换已满升级
- 15 秒超时自动关闭
- Space 键购买（需走进商品范围 + 金币足够）

## [0.3.0] - 2026-05-23

### Phase 3 — 道具系统 + 波次管理器
- 新增 12 种道具：HEART、COIN、NICKEL、LIFE、SPREAD、RAPIDFIRE、SHOTGUN、SPEED、NUKE、TELEPORT、SKULL、SHERRIFF
- 道具浮动动画 + 15 秒消失倒计时 + 消失前闪烁
- 波次管理器：难度曲线、敌人类型解锁、生成队列
- 敌人死亡掉落：5% 道具、10% 金币、85% 无掉落

## [0.2.0] - 2026-05-22

### Phase 2 — 敌人系统（Orc / Ogre / Mushroom）
- 新增 Enemy 类，单类 + type 字段设计
- 3 种地面敌人：Orc（基础）、Ogre（高血）、Mushroom（高速）
- AI 行为：每 20 tick 重新选择目标，25% 概率随机游走

- 受伤无敌帧（1.5 秒）、死亡重生（3 秒无敌）
